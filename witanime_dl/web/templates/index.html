📖 How to use

1. Select your download folder (where your Naruto episodes are saved).
2. Set the episode range (default 61–500 covers all of Shippuden).
3. Click Start — the app scans your folder and only collects links for episodes you're missing.
4. When done, click Download episodes.txt.
5. In FDM: File → Import → From Text File → episodes.txt

Refresh mode: If your old links expired, paste them in Refresh mode to get new permanent links.
